Colour palette

Make sure that colour combinations are of sufficient contrast to meet accessibility standards.

Primary colour palette

These are the only colours that our logo should appear in. This is because we are building recognition through repetition. All communications from the department need to use these as the main accent colours.

MHCLG Teal #00625E
Black #000000
White #ffffff

Core identity secondary colours (dark)

The secondary colour palette is made up of two sets of colours (darks and brights) to give a wider selection of use. These colours should be used to support the primary palette and create different levels of signposting where needed.

All dark colours are fully accessible to use for text colour.
